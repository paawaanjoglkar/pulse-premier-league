# External Libraries

## SheetJS (xlsx.min.js)

This directory should contain the SheetJS library for Excel export functionality.

### Download Instructions

1. Visit: https://github.com/SheetJS/sheetjs
2. Download the latest release of `xlsx.full.min.js`
3. Rename to `xlsx.min.js`
4. Place in this directory (`lib/xlsx.min.js`)

### Alternative CDN (for development only)

You can also use a CDN version temporarily:

```html
<script src="https://cdn.sheetjs.com/xlsx-0.20.1/package/dist/xlsx.full.min.js"></script>
```

**Important:** For production/offline use, download and bundle the library locally.

### File Size

Approximately 800-900KB minified.

### Version

Recommended: 0.20.1 or later
